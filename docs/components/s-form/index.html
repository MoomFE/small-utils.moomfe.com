# s-form 表单

Vuetify 的 v-form 组件的上层封装

  • W3C 标准中有此规定 (opens new window): 当一个 form 元素中只有一个输入框时, 在该输入框中按下回车应提交该表单
  • 该组件的目的是阻止该默认行为
